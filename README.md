# About

Одностраничный лендинг-визитка для frontend-разработчика с навигацией через резиновых уток.

## Стек

- React 18
- Vite 5
- Tailwind CSS 3
- Framer Motion 10
- canvas-confetti

## Запуск

```bash
# Установить зависимости (нужен Node.js, например через asdf: asdf local nodejs 20.10.0)
npm install

# Режим разработки
npm run dev

# Сборка
npm run build

# Превью сборки
npm run preview
```

## Структура

- **Hero** — имя, специализация, elevator pitch
- **Утки** — 4 основные (О себе, Астра, Билайн, Контакты) + секретная; на десктопе по углам экрана, на мобилке — FAB-меню
- **Модалки** — контент по клику на утку; закрытие по крестику, клику вне или ESC
- **Фон** — градиенты и плавающие пузыри

## Адаптив

- **Desktop (md+)** — утки в углах, секретная утка внизу по центру
- **Mobile** — одна кнопка-утка (FAB), по нажатию открывается меню с пунктами (О себе, Астра, Билайн, Контакты, Секрет)

## Доступность

- Утки с `role="button"` и `aria-label`
- Модалки с `role="dialog"`, `aria-modal="true"`, фокус при открытии и закрытие по ESC
- Учёт `prefers-reduced-motion` — анимации минимизируются
